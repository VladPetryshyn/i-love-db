#+title: Freifahrschein


* Motivation und Zweck

Ich habe mich bewusst für einen Anbieter entschieden, der mein Deutschlandticket als .pkpass-Datei für mein Wallet bereitstellt.

Das Deutschlandticket ist gemäß den Tarifbestimmungen ein personengebundenes, digitales Ticket, das in Verbindung mit einem amtlichen Lichtbildausweis gültig ist und nicht etwa erst durch die Nutzung einer spezifischen, oft fehleranfälligen und auf Netzwerkverbindung angewiesenen Drittanbieter-App, die ein kleines Logo über den Bildschirm bewegt.

Leider sieht die Realität bei Kontrollen oft anders aus: Zugpersonal oder lokale Verkehrsverbände fordern regelmäßig eben diese Animation ein. Ich bin es leid, ständig mein Recht zu erstreiten, die Polizei für Belanglosigkeiten hinzuzuziehen oder gar Bußgelder zu riskieren, nur weil mein Ticket nicht den persönlichen Vorlieben oder technischen Unzulänglichkeiten des Kontrollpersonals entspricht.

Diese App ist meine Antwort darauf: 
Eine einfache, zuverlässige Lösung, die genau das tut, was nötig ist, um sich bei einer Kontrolle nicht zum „Affen“ machen zu müssen und die Bedürfnisse der outgesourced und lohngedumpten DienstleisterInnen zufrieden zu stellen.

* Funktionsweise & Features
Die App wurde entwickelt, um maximale Unabhängigkeit und Datenschutz zu gewährleisten:

    Kein Store-Zwang: Funktioniert als Progressive Web App (PWA) direkt im Browser deiner Wahl.

    100% Offline: Nach dem ersten Laden der Webseite wird die App lokal auf deinem Gerät gespeichert. Es ist zu keiner Zeit eine aktive Internetverbindung für die Anzeige erforderlich. Deine Daten und Bilder verlassen niemals dein Endgerät. 

    Anzeige: Du kannst ein beliebiges Bild (z. B. deinen UIC- oder VDV-Barcode) hinterlegen.

    Animation: Um den Erwartungen des Kontrollpersonals an eine „aktive“ App zu entsprechen, lässt sich ein zweites Bild (z. B. ein Logo) flüssig hin- und herbewegen.


* Rechtlicher Hinweis:
    Diese App ist kein Ersatz für ein gültiges Ticket und muss in Verbindung mit einem solchen verwendet werden.

    Ich übernehme ausdrücklich keinerlei Haftung für die Nutzung dieser App – auch dann nicht, wenn ein gültiges Ticket vorliegt. Die Nutzung erfolgt auf eigene Gefahr und Verantwortung. Jegliche Konsequenzen, die aus der Vorlage dieser App bei einer Kontrolle entstehen, tragen die Nutzer selbst.

    function sanitizeNumber(value, min, max, defaultValue) {
        let num = parseInt(value, 10);
        if (isNaN(num)) return defaultValue;
        return Math.max(min, Math.min(max, num));
    }

    function sanitizeText(text) {
        if (!text) return '';
        return text
            .replace(/<script\b[^<]*(?:(?!<\/script>)<[^<]*)*<\/script>/gi, '')
            .replace(/on\w+\s*=/gi, 'X_on_');
    }

    function processFile(inputElement, storageKey) {
        const file = inputElement.files[0];
        if (file) {
            const reader = new FileReader();
            reader.onload = (e) => localStorage.setItem(storageKey, e.target.result);
            reader.readAsDataURL(file);
        }
    }
